# Cadatro de usuários

> ## Caso de sucesso
1. Sistema recebe os dados do usuários
2. Sistema valida os dados recebidos
3. Sistema registra os dados na tabela de usuários

1. Sistema retorna uma mensagem de erro

> ## Exceção - Erro ao registrar os dados
1. Sistema retorna uma mensagem de erro
